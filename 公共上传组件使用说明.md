# 公共上传组件使用说明

## 概述

公共上传组件 (`Upload`) 是一个功能强大、可复用的文件上传组件，支持拖拽上传、文件选择、进度显示、错误处理等功能。

## 功能特性

### 基础功能
- ✅ 拖拽上传支持
- ✅ 点击选择文件
- ✅ 文件类型验证
- ✅ 文件大小限制
- ✅ 多文件上传支持
- ✅ 文件预览（图片、视频）

### 状态管理
- ✅ 文件上传状态（待上传、上传中、成功、失败）
- ✅ 上传进度显示
- ✅ 错误信息展示
- ✅ 文件列表管理

### 用户体验
- ✅ 实时文件预览
- ✅ 文件信息显示
- ✅ 上传结果反馈
- ✅ 可删除已选择文件
- ✅ 重试上传功能

### 可配置选项
- ✅ 支持的文件类型
- ✅ 最大文件大小
- ✅ 是否支持多文件
- ✅ 自定义上传处理函数
- ✅ 自定义样式

## 基本用法

### 最简单的使用方式

```tsx
import Upload from '@/components/common/Upload'

function MyComponent() {
  return (
    <Upload
      accept="image/*"
      onFileSelect={(files) => console.log('Selected files:', files)}
    />
  )
}
```

### 视频上传示例

```tsx
import Upload from '@/components/common/Upload'

function VideoUpload() {
  const handleUpload = async (file: File) => {
    // 自定义上传逻辑
    console.log('Uploading video:', file.name)
    // 这里可以调用 API 上传文件
  }

  return (
    <Upload
      accept="video/*"
      maxSize={500} // 500MB
      multiple={false}
      uploadText="拖拽视频文件到此处"
      buttonText="选择视频文件"
      onFileSelect={(files) => console.log('Selected videos:', files)}
      onUpload={handleUpload}
    />
  )
}
```

### 图片上传示例

```tsx
import Upload from '@/components/common/Upload'

function ImageUpload() {
  const handleUpload = async (file: File) => {
    // 上传图片到服务器
    const formData = new FormData()
    formData.append('image', file)
    
    const response = await fetch('/api/upload', {
      method: 'POST',
      body: formData
    })
    
    if (!response.ok) {
      throw new Error('上传失败')
    }
    
    return response.json()
  }

  return (
    <Upload
      accept="image/*"
      maxSize={10} // 10MB
      multiple={true}
      maxFiles={5}
      uploadText="拖拽图片到此处或点击选择"
      buttonText="选择图片"
      autoUpload={true}
      onUpload={handleUpload}
      onError={(error) => console.error('Upload error:', error)}
    />
  )
}
```

## 高级用法

### 自定义上传处理

```tsx
import Upload from '@/components/common/Upload'

function CustomUpload() {
  const customUpload = async (file: File) => {
    // 自定义上传逻辑，比如分片上传、压缩等
    const chunks = await splitFile(file)
    
    for (const chunk of chunks) {
      await uploadChunk(chunk)
      // 更新进度
    }
  }

  return (
    <Upload
      accept="*"
      customUpload={customUpload}
      beforeUpload={(file) => {
        // 上传前验证
        return file.size < 100 * 1024 * 1024 // 100MB
      }}
    />
  )
}
```

### 自定义文件预览

```tsx
import Upload from '@/components/common/Upload'

function PreviewUpload() {
  const renderPreview = (file: UploadFile) => {
    if (file.type.startsWith('image/')) {
      return (
        <img 
          src={file.preview} 
          alt={file.name} 
          style={{ maxWidth: '100px', maxHeight: '100px' }}
        />
      )
    }
    return <div>{file.name}</div>
  }

  return (
    <Upload
      accept="*"
      renderPreview={renderPreview}
    />
  )
}
```

## API 参考

### UploadProps

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `accept` | `string` | - | 支持的文件类型，如 'image/*', 'video/*', '.pdf' 等 |
| `maxSize` | `number` | `10` | 最大文件大小限制（MB） |
| `multiple` | `boolean` | `false` | 是否支持多文件上传 |
| `disabled` | `boolean` | `false` | 是否禁用上传 |
| `onFileSelect` | `(files: File[]) => void` | - | 文件选择回调 |
| `onUpload` | `(file: File) => Promise<void>` | - | 文件上传回调 |
| `onError` | `(error: string) => void` | - | 上传错误回调 |
| `uploadText` | `string` | `'拖拽文件到此处或点击选择'` | 上传区域文本 |
| `buttonText` | `string` | `'选择文件'` | 选择文件按钮文本 |
| `className` | `string` | - | 自定义样式类名 |
| `dragActiveClassName` | `string` | - | 拖拽激活时的样式类名 |
| `showFileList` | `boolean` | `true` | 是否显示文件列表 |
| `maxFiles` | `number` | - | 最大文件数量限制 |
| `renderPreview` | `(file: UploadFile) => React.ReactNode` | - | 自定义文件预览组件 |
| `autoUpload` | `boolean` | `false` | 是否自动上传 |
| `beforeUpload` | `(file: File) => boolean \| Promise<boolean>` | - | 文件上传前的验证函数 |
| `customUpload` | `(file: File) => Promise<void>` | - | 自定义上传处理函数 |
| `children` | `React.ReactNode` | - | 自定义上传区域内容 |

### UploadFile

| 属性 | 类型 | 说明 |
|------|------|------|
| `id` | `string` | 文件唯一标识 |
| `file` | `File` | 原始文件对象 |
| `name` | `string` | 文件名 |
| `size` | `number` | 文件大小（字节） |
| `type` | `string` | 文件类型 |
| `preview` | `string` | 文件预览 URL |
| `status` | `'pending' \| 'uploading' \| 'success' \| 'error'` | 上传状态 |
| `progress` | `number` | 上传进度（0-100） |
| `error` | `string` | 错误信息 |


### 4. 上传进度反馈

```tsx
<Upload
  onUpload={async (file) => {
    // 显示上传进度
    showProgress(true)

      await uploadFile(file)
      showSuccess('上传成功')
    } catch (error) {
      showError('上传失败')
    } finally {
      showProgress(false)
    }
  }}
/>
```

## 注意事项

1. **文件预览**：图片文件会自动生成预览，视频文件目前只显示图标
2. **内存管理**：大文件的预览可能会占用较多内存，建议合理设置文件大小限制
3. **并发控制**：同时上传多个文件时，注意服务器的并发处理能力
4. **错误恢复**：实现重试机制时，注意避免无限重试
5. **用户体验**：提供清晰的上传状态反馈和错误信息

## 扩展建议
